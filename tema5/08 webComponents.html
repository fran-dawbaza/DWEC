Componentes web o web components

Los web components son un conjunto de estándares con el propósito de generar componentes autónomos para utilizar como elementos de nuestros sitios web

Para su diseño utilizaremos cuatro API

  • Custom Elements
  • Shadow DOM
  • Templates
  • JS modules

Para comienzar os invito a revisar la web https://es.javascript.info/web-components
