Una petición AJAX (Asynchronous JavaScript And XML) consiste en realizar una petición de datos a algún servidor para utilizar los datos de la respuesta normalmente en la construcción dinámica de algún bloque de código HTML de la página cargada. Es decir, vamos a modificar parte de la página web visualizada sin necesidad de cargar la página completamente

Vamos a usar un servidor para realizar una petición y obtener una respuesta
