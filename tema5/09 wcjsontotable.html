Vamos a hacer un componente para construir una tabla a partir de un objeto JSON

Ideas para implementar:

  • Calendario personalizado, con opciones: de poner hoja de estilo por atributo; tipo de calendario (español/inglés); de poner fecha para mostrar su mes; **de añadir eventos
  • Carrusel de imágenes, permitiendo: indicar el nombre del objeto JS que contiene las imágenes a motrar; añadirle subetiquetas que especifiquen la ruta de cada imagen, enlace, comentario; indicar también atributos para estilos: alto, ancho, ajuste de imágenes...
  • Tarjetas (Cards), permitiendo: elementos principales como imagen, título, descripción, enlace y estilos
  • Select
  • Barra progreso
  • Cambio de estilos (Nocturno/diurno/aumento o reducción de tamaño fuente ...)
  • Paginador de tablas grandes
