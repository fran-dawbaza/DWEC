import, export, module

Vamos a ver cómo trabajar con programación modular en javascript

Puedes acceder a la documentación en los enlaces:

  • import: https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Sentencias/import
  • export: https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/Sentencias/export
  • module: https://developer.mozilla.org/es/docs/Web/JavaScript/Guide/M%C3%B3dulos

A modo de resumen, debes tener presente tres puntos principales para que funcione la importación de módulos:

  • Usar type="module" en la etiqueta script para indicar que usaremos módulos
  • Usar export en las variables, constantes o funciones que queramos exportar desde el archivo .js que queramos importar
  • Usar import del archivo .js

Esta misma página contiene un ejemplo de utilización, vamos a importar la constante "diccionario" que es un array de más de un millón y cuarto de palabras
